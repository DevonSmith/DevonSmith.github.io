Instructions

This is a reverse polish notation, or postfix calulator. Postfix or reverse polish notation is a mathematical notation in which the operators follow the operands. This calulator will perform the following operations:

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/)

The use of postfix notation means that there is no need for parenthesis in mathematical expressions. This calculator can process mathematical strings using only numbers along with +, - , *, and / symbols.

Example

The following is an example of an expression that can be expressed in postfix notation. This example comes from wikipedia.

The expression ((15 / (7 − (1 + 1))) * 3) − (2 + (1 + 1)) can be expressed with postfix notation as 15 7 1 1 + - / 3 * 2 1 1 + + -

This type of notation was commonly used in calculators because it was simple to implement using a basic stack. This projects attempts to replicate that behavior using Javascript. If you would like to see how the calculator is using the stack to solve a postfix expression you can check the "Show stack" checkbox and you will get a list of the stack elements after each operation.
